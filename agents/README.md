# Agents

This directory will hold `.agent.md` files for content-creation agents when the workflow warrants them.

Potential future agents:
- `content-carousel-writer` — generates carousel scripts from topic + persona
- `content-caption-writer` — generates captions from carousel scripts
- `content-image-prompter` — generates Gemini/DALL-E prompts from carousel themes
- `content-decision-logger` — tracks content strategy decisions
