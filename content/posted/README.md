# Posted Content

This folder contains all content that has been published on Instagram. Each post has its own folder with all assets and metadata.

## Structure

Each posted item is in a subfolder named `XX_slug/` containing:
- `post_record.md` — metadata (date posted, platform, what version was used, performance notes)
- Original carousel, caption, and image prompt files
- Any alternate versions that were posted
