# Content Workspace

This folder contains production-ready Instagram assets extracted from `content/instagram_carousel_playbook.md`.

## Folders

- `carousels/`: six standalone 9-slide scripts.
- `captions/`: caption assets mapped to each carousel.
- `image_prompts/`: Gemini cover prompts, style guide, and inner-slide prompt.

## Workflow

1. Pick a script from `carousels/`.
2. Generate visual assets from `image_prompts/`.
3. Publish with matching caption in `captions/`.
4. Track strategy changes in `decisions/decision_log.md`.
