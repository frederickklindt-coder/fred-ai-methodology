# Global Visual Style Guide

Use this style consistently across all carousel images.

- Format: 1080x1350 (4:5)
- Background: dark blue-black (`#0D1117`)
- Accent colors: electric blue (`#58A6FF`), bright green (`#3FB950`), soft white (`#E6EDF3`)
- Visual language: grids, nodes, terminal UI, architecture flows
- Tone: modern, calm, competent, minimal
- Avoid: stock-photo faces/hands, logos, watermarks

---

## Frederick Avatar Integration

When the Frederick caricature avatar appears in carousel slides or covers:

- **Avatar is always warm-toned** (golden highlights, warm skin, burgundy accents) — contrasts against the cool dark tech background
- **Robot companion** (silver-white, blue LED eyes `#58A6FF`) appears alongside the avatar
- **Brain hologram** (electric blue `#58A6FF` wireframe) ties into the existing accent palette
- Avatar covers use a **cosmic background** variant (deep navy-purple with stars) instead of pure `#0D1117` — acceptable for covers only
- Inner slides keep `#0D1117` background; avatar appears as a small bust/icon element if used
