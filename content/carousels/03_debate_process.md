# 03 - Prompt Debate Ship

## Topic
How to compare model outputs and make stronger decisions.

## Audience
Developers who only use one model today.

## Result
A multi-model decision framework.

## CTA Keyword
`DEBATE`

## Slides

### Slide 1 - Hook
I never trust the first AI answer.
I make models compete.

### Slide 2 - Raw Context
Start with a context dump:
notes, constraints, goals, edge cases.

### Slide 3 - Double Send
Send the same prompt to two models.

### Slide 4 - Debate
Challenge assumptions.
Ask for evidence.
Push back on generic outputs.

### Slide 5 - Select
Pick the strongest reasoning per section, not per model.

### Slide 6 - Log
Record the decision with ID, owner, status, and impact.

### Slide 7 - Myth Break
There is no universal best model.
There is only best-fit for this task and context.

### Slide 8 - Proof
In one architecture decision:
- One model found docs.
- One model found payload risk.
Combined output shipped.
