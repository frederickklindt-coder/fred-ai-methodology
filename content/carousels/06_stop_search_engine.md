# 06 - Stop Search Engine Mode

## Topic
Why most people underuse AI and how to level up.

## Audience
Anyone using AI as single-shot Q&A.

## Result
A practical loop for production-grade outputs.

## CTA Keyword

### Slide 1 - Hook

### Slide 3 - Builder Pattern
Context dump -> draft -> debate -> revise -> second model -> decide -> implement.

### Slide 4 - Context Principle
Output quality scales with context quality.

### Slide 5 - Debate Principle
Challenge assumptions and request evidence.

### Slide 6 - Version Principle
Treat prompts like code: V1, V2, V1.7.

### Slide 7 - Myth Break
Great prompting is iterative, not magical first-try perfection.

### Slide 8 - Proof
Multi-model comparison catches different failure modes.
